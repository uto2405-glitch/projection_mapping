## 현장 모드 (두 기기)

### 최초 설치 — 개관 전 1회

1. 출력 PC ↔ 프로젝터 HDMI 연결, 프로젝터 물리 위치·줌 고정
2. 기동 스크립트 1회 실행 — 서버·릴레이가 뜨고 키오스크 브라우저가
   `?role=output` 전체화면으로 자동 오픈 (터미널·주소창 없음, Q14)
3. 같은 스크립트를 OS 시작 프로그램에 등록 — 이후 전원 ON = 전시 시작
4. 출력 화면의 QR을 아이패드 카메라로 스캔 → 사파리에 드로잉 UI(`?role=draw`) 열림, 릴레이 자동 접속
5. 아이패드에서 정렬 모드 진입 → 귀퉁이 4개를 원격 드래그해 실물 표면에 핀
   — 이 1~5단계 전체가 채점표 B4의 측정 대상: 3분 이내
6. 정렬값은 출력 쪽 `ldp:corners`에 자동 저장 (다음 가동 시 복원)

### 일상 가동 — 매일 (두 동작)

1. PC 전원 ON — 서버·릴레이·키오스크 출력까지 자동으로 뜨고 정렬 자동 복원
2. 아이패드 QR 스캔 → 바로 드로잉 가능
(프로젝터가 물리적으로 밀렸을 때만 정렬 모드로 재정렬)

### 라이브 운용

- 펜: 색 · 굵기 · 지우개
- 연출: 글로우 토글 · 잔상 슬라이더(2~30초, 별도 영구 모드)
- 세션 교체(관람객/장면 전환): 모두 지우기
- 기념·기록: PNG 내보내기 1버튼

### 장애 대응

- 아이패드 연결 끊김: 사파리 새로고침 → QR 재스캔 (릴레이 생존 시 즉시 복귀)
- 출력 창 이탈: 기동 스크립트 재실행 (또는 `?role=output` 재진입) → 정렬값 자동 복원
- 정렬이 크게 틀어짐: 정렬 모드에서 리셋 후 재핀

## 책상 실험 모드 (한 기기)

1. `npm run dev` → 브라우저 창 두 개: `?role=draw`, `?role=output`
2. BroadcastChannel 자동 연결 — 릴레이·QR·정렬 불필요(전체화면 판정)
3. 채점: `cd grader && npm run grade` (공식) / `grade:quick` (참고)

## 제외 (로드맵 — 채점표 개정 전 구현 금지)

접속 코드 · 정렬 멀티 프리셋 · 멀티 서페이스 · 자동 캘리브레이션 (SCORECARD C3 준용)
