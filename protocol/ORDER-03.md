## 범위

- **4코너 워프**: 호모그래피 행렬 1개(역사상 워프, 프래그먼트 셰이더) — 합성
  텍스처를 실물 표면 한 면에 핀. 범위 밖은 검정.
- **원격 정렬 (Q7)**: draw 페이지 `[data-test="align-mode"]` 진입 → 코너 핸들 4개
  드래그 → 동기화 계층(획과 같은 채널)으로 출력에 전달 → 출력이 적용·저장
  (`ldp:corners`, A6 불변). 리셋 버튼 포함.
- **ws 릴레이 (두 기기)**: `npm run relay` → ws://<host>:8787 (Node ws). 동기화
  추상 계층에 ws 전송 추가 — localhost 접속(책상·채점)에서는 시도조차 하지 않아
  콘솔 무오염(A4), LAN IP 접속에서 자동 활성.
- **QR 접속 (WORKFLOW 최초 설치 4)**: 출력 화면 구석에 draw URL QR — 드로잉
  클라이언트 활동이 감지되면 자동 숨김, 유휴 시 재표시.
- **원클릭 기동 (Q14)**: `scripts/start-field.bat`(윈도) · `scripts/start-field.command`(맥)
  — 서버+릴레이 기동, LAN IP로 키오스크 출력 자동 오픈. OS 자동시작 등록법 README 기재.
