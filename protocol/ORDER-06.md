# 발주서 #6 — 미디어 레이어 · 갤러리 영속화 · 센서 리액티브 (2026-07-17 발주)

발주 근거: SCORECARD 개정 3호 (사람 승인 "모든 구현을 승인" + "제한 없이 마음껏").
VISION #4·#5·#7의 브라우저 검증 가능 조각을 한 사이클로 묶는다. 프로젝터·실물 없이
전부 책상(브라우저)에서 시험 가능해야 한다는 사용자 제약을 유지한다.

## 범위

### A. 미디어 레이어 (VISION #4)
- draw 페이지에서 이미지·영상 파일 로드 → 출력 면에 워프 적용되어 합성
  (가산 합성 — 프로젝터 광학: 미디어 + 잉크 + 글로우)
- 변환 조작: 이동(미디어 모드 드래그)·크기·회전·불투명도, 토글 ON/OFF
- 전송: 같은 기기 BroadcastChannel(Blob 구조 복제), 두 기기 ws 청크(48KB, ≤25MB)
- 영상은 음소거 루프 재생. 재생 중엔 연속 제시(비디오 텍스처)
- 기본 OFF — 채점 경로 불변. 출력 리로드 시 draw가 재전송(announce 편입)

### B. 갤러리 영속화 (VISION #7 테제 5 기반)
- draw 토글 `[data-test="gallery-save"]` ON이면 완성 획을 릴레이가 JSONL로 영속화
- 출력(재)접속 시 릴레이가 저장 획 리플레이 — 시스템 전체 재부팅에도 작품 유지
- 모두 지우기 = 파일 초기화, 실행취소 = 해당 획 제거 (운영자 중재)
- 책상(BC 전용) 모드에선 비활성 (릴레이 필요) — 배지로 안내

### C. 센서 리액티브 (VISION #5 — 모션 기반 1차 경로)
- `?role=sensor`: 출력 PC 웹캠 프레임 차분 → 모션 중심·강도 (ML 없음, 전부 로컬 — C1)
- 버스 발행 {t:'motion'} → 출력이 모션 지점에 별 반짝임 스폰 — "손을 따라다니는 별"
- 랜딩 허브 링크 + 미리보기(모션 오버레이) 포함
