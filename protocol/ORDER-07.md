# 발주서 #7 — 멀티 아웃풋 · 다중 동시 드로잉 · 필압 (2026-07-17 발주)

발주 근거: SCORECARD 개정 4호 (사람 승인 "모두 승인 … 무한으로 개선").
C3 마지막 잔여의 브라우저 검증 가능 조각 전부. 자동 캘리브레이션만 승격 단계 유보.

## 범위

1. **멀티 아웃풋 (#6-lite, VISION 테제 4)** — `?role=output&id=n` 클라이언트 N개가
   같은 버스를 구독하고 각자 자기 워프만 렌더. 워프 저장 `ldp:corners:{id}`·`ldp:warp:{id}`.
   정렬 UI에 대상 출력 선택(`[data-test="align-target"]`) — 워프 메시지에 out 태그.
   기본 출력(id 없음)은 기존 키·기존 동작 그대로 (A6 불변). 마스크·엣지 블렌딩은
   물리 겹침이 필요하므로 승격 단계 카드.
2. **다중 사용자 동시 드로잉** — 출력이 획을 발신자(sid)별 네임스페이스로 키잉:
   여러 draw 클라이언트의 s000이 충돌하지 않는다. 레지스트리·렌더 마크는 공개 id
   유지(채점 불변). 클라이언트별 자동 파스텔 배색(저장된 취향 우선).
3. **압력 감지 굵기** — 애플펜슬 필압을 포인트별 폭 계수로 (0.4×~1.7×, 스무딩 조각별
   반영). 필압 미지원 입력(마우스·합성 이벤트 압력 0)은 계수 1 — 채점 시각 경로 불변.
