# VISION — 확장 비전과 장기 로드맵

지위: 방향 문서다. 채점표가 아니다 — 여기 적힌 것은 각자의 사이클에서
채점표 개정 게이트를 통과하기 전까지 구현 금지(C3 준용).
확장 아이디어는 채점표를 건드리지 말고 이 문서에 먼저 적는다.

## 종착지

관람객을 둘러싸는 다면 몰입 프로젝션 공간
(레퍼런스: protocol/anchors/anchor-01.png — 2026-07-16 제공).
핵심 불변 원칙: **아이패드에서 만진 것이 즉시 공간에 반영된다.**
콘텐츠는 획(드로잉)에서 시작해 미디어 레이어, 센서 반응형 연출,
관객이 남기는 지속 상태로 확장된다.

## 아키텍처 테제

1. **메시지 버스가 본체.** 모든 입력원(draw, sensor, visitor, 미디어 컨트롤)은
   발행자이고 출력은 구독자다. 새 기능은 렌더러 수술이 아니라 발행자 추가로 들어온다.
2. **렌더는 "콘텐츠 → 오프스크린 텍스처 → 워프 메시".** 면을 늘리는 일이
   메시를 추가하는 일이 되도록. (#3 설계 씨앗 — ORDER-03 작성 시 편입)
3. **센서는 갈아끼우는 부품.** 1차 경로는 출력 PC의 웹캠/뎁스캠 +
   브라우저 내 포즈·손 추정(로컬 연산, C1 준수). 아이패드 LiDAR의 웹 직접
   접근은 사파리에서 현재 불가 — 필요해지면 네이티브 브리지를 별도 사이클로.
4. **출력은 복수다.** 프로젝터 N대 = `?role=output&id=n` 클라이언트 N개.
   같은 버스를 구독하고 각자 자기 워프·마스크만 렌더한다.
   캘리브레이션은 id별 저장(`ldp:corners:{id}`).
5. **관객 상태는 릴레이가 영속화한다.** 획은 증발하지만 깃발은 쌓인다 —
   이벤트를 파일로 남기고 출력 부팅 시 재생한다.

## 로드맵 (착수 전 채점표 개정 게이트 필수)

- **#4 미디어 레이어** — 이미지·영상을 면에 입히고 드로잉과 합성.
  C3 "레이어/이미지 불러오기" 해제 개정 동반.
- **#5 센서·리액티브** — `?role=sensor` 클라이언트 신설, 포즈 → 파티클.
  손을 흔들면 별 반짝임이 손을 따라다니는 연출. 추적 지연·프레임 등
  새 A항목 신설 필요.
- **#6 멀티 서페이스·멀티 프로젝터** — C3 해제 개정 동반.
  · 워프는 4코너(2×2 제어점)의 일반화인 N×M 격자 메시 — 제어점 드래그로 곡면 손정렬
  · 출력별 마스크 다각형("각 영상의 모서리 지정")
  · 겹침 구간 알파 페더링 + 밝기 매칭 = 엣지 블렌딩
  · 곡면 한계선: 격자 손정렬로 이음매 품질이 안 나오면 3D 모델 기반 투영
    (0단계 Q3의 세 번째 길 — 대상 지오메트리 + 프로젝터 캘리브레이션 →
    텍스처 자동 분배)을 #6 후반 카드로 연다
  · 대표 케이스: 지구본 구면 맵핑, 사방 3~4대
    (protocol/anchors/case-globe.png — 2026-07-16 제공)
- **#7 관객 참여·지속 상태** (신설)
  · `?role=visitor` 제한 UI — 예: 지도에서 자기 나라 선택만 가능
  · 지속 상태 레이어: 릴레이 파일 영속화 + 부팅 시 재생 (테제 5)
  · 국가→구면 좌표 테이블로 깃발 렌더 — "방문객 깃발꽂기" 케이스
  · 운영자 중재: 전체 초기화 · 개별 삭제 (장난 입력 대응)
  · 고급 경로: 지구본 주변 센서로 실물 직접 터치 감지 — #5 성숙 후
  · 새 A항목 후보: 깃발 반영 지연 · 영속 무결성(재부팅 후 개수 일치)

## 활용 시나리오 (발주 설계·영업 자료 양용)

MVP(#1~#3)로 가능:
- **도슨트 라이브 주석** — 해설하는 순간 유물 위에 빛으로 밑줄·화살표.
  잔상 8초가 자동 정리. 빛은 실물에 무해. 1순위 파일럿 후보.
- **어린이 낙서벽** — 잔상 영구 모드 + 모두 지우기로 관람객 교체 운영,
  PNG 내보내기가 기념품. (동시 다중 드로잉은 C3 제외 — 순번제)
- **영업 시연 킷** — "정렬 3분"은 곧 미팅 중 설치 완료.
  아이패드 + 미니 프로젝터 가방이 수주 도구.

#5 이후:
- **인터랙티브 포토존** — 손을 따라다니는 별 반짝임(취향 앵커의 그 장면).
  관람객이 찍어 올리는 사진이 곧 홍보가 되는 구조.
